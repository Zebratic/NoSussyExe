 An automatic .NET dumper for executables that you might not trust.

## How to use
1. Drag & Drop any executable onto the build exe.
2. Enjoy.

## Notice
- THE DUMPER WORKS FOR .NET ONLY!

## Features
- .NET Dumper
- Native Dumper
- System Dumper
- Thread Suspension
